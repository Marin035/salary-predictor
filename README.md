# 🤖 AI Salary Predictor

Прост Full-Stack проект, който демонстрира използването на Machine Learning модел през уеб интерфейс.

## 🚀 Технологии
* **Python** & **Scikit-learn** - за трениране на модела (Линейна регресия).
* **FastAPI** - за създаване на високопроизводително API.
* **HTML/JavaScript** - за потребителския интерфейс.

## 🛠️ Как се стартира?
1. Инсталирайте нужните библиотеки: `pip install fastapi uvicorn scikit-learn numpy joblib`
2. Стартирайте сървъра: `python -m uvicorn main:app --reload`
3. Отворете `index.html` в браузъра си.
